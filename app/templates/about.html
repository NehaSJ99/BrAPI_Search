About BrAPI Search Tool

The BrAPI Search Tool, developed by the GrainGenes team, leverages the BrAPI Interface to provide a unified platform for searching various data types, including germplasm, traits, and trials across multiple databases.

About GrainGenes

GrainGenes is a comprehensive database for Triticeae and Avena, providing extensive molecular and phenotypic information for wheat, barley, rye, and oat. It serves as a central data repository and facilitates community activities within small grains research.

About BrAPI

BrAPI is a standardized RESTful web service API specification designed for communicating plant breeding data. It enables seamless data sharing across different databases and tools involved in plant breeding.

To access the code and scripts for this project, please visit our GitHub.
