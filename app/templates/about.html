About BrAPI Search Tool

GrainGenes, a database for Triticeae and Avena, is a comprehensive resource for molecular and phenotypic information for wheat, barley, rye, and oat. It is is a digital platform that serves small grains research communities as their central data repository and as a facilitator for community activities.

BrAPI is a standardized RESTful web service API specification for communicating plant breeding data. BrAPI allows for easy data sharing between databases and tools involved in plant breeding.

BrAPI Search Tool by the GrainGenes team provide a platform to search data types such as germplasm, traits and trials in various databases by leveraging Brapi Interface.

To access code and scripts, please visit GitHub.
