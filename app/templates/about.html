About BrAPI Search Tool

GrainGenes, a database for Triticeae and Avena, is a comprehensive resource for molecular and phenotypic information for wheat, barley, rye, and oat.

BrAPI is a standardized RESTful web service API specification for communicating plant breeding data

To access code and scripts, please go to GitHub.
